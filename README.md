# LINE 貼圖裁切工具（Web）\n\n本專案為一款 **Web 版 LINE 貼圖裁切與規格校驗工具**（Sticker Cropping & Validation Tool）。目標是把 LINE 貼圖的官方製作準則轉化為「可執行的程式邏輯」與「可見的 UI 限���」，讓使用者匯出時就能得到 **符合上架規範** 的圖片與素材包。\n\n---\n\n## 核心目標\n\n- 讓使用者專注創作，工具負責規格校對與自動化處理\n- 產出圖片時即完成尺寸、格式、檔案大小等規範驗證\n- 提供裁切安全區、留白提示、去背提醒等「降低退件率」的 UI\n\n---\n\n## 功能規劃（對應規格 → 程式邏輯 / UI）\n\n### 1) 精準尺寸模板與自動縮放\n內建三種模式：\n\n- **貼圖圖片模式**：最大寬 **370px** × 最大高 **320px**\n- **主要圖片模式**：固定 **240px × 240px**\n- **聊天室標籤模式**：固定 **96px × 74px**\n\n**輸出尺寸限制：**\n- 輸出的寬與高會強制為 **偶數**（避免縮放時因取樣造成的失真或對齊問題）\n\n### 2) 智慧裁切與留白（安全區）\n- 在裁切介面顯示「安全區」框線\n- 依準則建議，裁切後圖案與邊框保留約 **10px 留白**\n\n**去背提醒：**\n- 上傳後檢查背景是否可能為不透明底\n- 若偵測到背景不透明，提供警告（可視需求加入簡易去背流程）\n\n### 3) 匯出規格自動校驗\n匯出時自動檢查：\n\n- **格式**：統一輸出 **PNG**\n- **色彩**：強制 **RGB**\n- **解析度**：確保 **72dpi 以上**\n- **檔案大小**：\n  - 每張壓到 **1MB 以下**\n  - 批次匯出時自動打包 **ZIP**，總大小不超過 **60MB**\n\n### 4) 內容合規性輔助（可先做規則版）\n- **字數限制提示**：\n  - 名稱 **40 字以內**、說明 **160 字以內**\n  - UI 即時計數，並提示全形字元可能以 2 個字元計\n- **違規提示（初階）**：\n  - 以關鍵字與簡單規則提醒可能的商標/不當內容\n  - 後續可擴充 OCR / 模型檢測\n\n---\n\n## Web 版技術建議（可依實作再調整）\n\n- 前端：React / Vue / Svelte 皆可\n- 影像處理：\n  - 瀏覽器端 Canvas（裁切、縮放、輸出 PNG）\n  - 或搭配 WASM 影像處理（若需要更進階去背/壓縮控制）\n- 壓縮與 ZIP：\n  - PNG 壓縮（依需求選用對應 JS 套件）\n  - ZIP 打包（例如 JSZip）\n- 規格驗證：匯出前在前端執行（立即回饋）\n\n---\n\n## 待補內容（下一步）\n\n- [ ] UI 流程圖（上傳 → 模式選擇 → 裁切 → 檢查 → 匯出）\n- [ ] 規格檢查清單（尺寸/偶數/透明度/大小/ZIP 總量）\n- [ ] Demo 截圖或 GIF\n- [ ] 專案結構與安裝方式\n\n---\n\n## 產出物\n\n- 單張 PNG（符合模式規格）\n- 批次輸出 PNG + ZIP 素材包（含檔案大小總量檢查）\n
